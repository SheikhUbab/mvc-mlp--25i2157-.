# Multilayer Perceptron (MLP) - MVC Project

## Overview
This project implements a Multilayer Perceptron (MLP) from scratch using NumPy.  
It covers forward propagation, loss calculation, backpropagation, and optimization techniques.

---

## Network Architecture

### Manual Implementation:
- Input Layer: 2 neurons
- Hidden Layer 1: 2 neurons (Sigmoid)
- Hidden Layer 2: 2 neurons (Sigmoid)
- Output Layer: 1 neuron (Sigmoid)

### MNIST Implementation:
- Input Layer: 784 neurons
- Hidden Layer 1: 128 neurons
- Hidden Layer 2: 64 neurons
- Output Layer: 10 neurons
